**📄 Descripción**
Sistema de consola para administrar distintos tipos de vehículos en una empresa de transporte. Permite registrar vehículos, visualizar su información y simular acciones de aceleración y frenado con comportamiento diferenciado según el tipo de vehículo.

**✅ Funcionalidades**
*Registrar diferentes tipos de vehículos
*Mostrar la información básica de cada vehículo
*Simular aceleraciones y frenadas
*Reflejar el cambio de velocidad tras cada acción en consola
*Comportamiento personalizado de acelerar/frenar según el tipo de vehículo
